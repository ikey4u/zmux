# Shortcut Quickstart Reproduction Guide

## Outputs

- `docs/diagrams/shortcut-quickstart.png`: Chinese quickstart cheat sheet
- `docs/diagrams/shortcut-quickstart-en.png`: English quickstart cheat sheet
- `docs/diagrams/generate_shortcut_quickstart.py`: reproducible generation script
- `docs/diagrams/shortcut-quickstart-philosophy.md`: visual style guide

## Single Source of Truth

- `docs/SHORTCUT.md` is the only source of truth for shortcut content.
- The images must be derived from `docs/SHORTCUT.md`; do not edit the PNG files manually.
- If `docs/SHORTCUT.md` adds new sections, commands, or behavioral notes, update the script mappings first, then regenerate the images.

## Dependencies

- Python 3
- Pillow

If Pillow is missing on your machine, run:

```bash
python3 -m pip install pillow
```

## Rebuild Directly

Run this from the repository root:

```bash
python3 docs/diagrams/generate_shortcut_quickstart.py
```

The script will generate both of these files:

- `docs/diagrams/shortcut-quickstart.png`
- `docs/diagrams/shortcut-quickstart-en.png`

It is also recommended to verify the output sizes:

```bash
python3 -c "from PIL import Image; from pathlib import Path; p1=Path('docs/diagrams/shortcut-quickstart.png'); p2=Path('docs/diagrams/shortcut-quickstart-en.png'); print(p1.name, Image.open(p1).size); print(p2.name, Image.open(p2).size)"
```

## Steps When `docs/SHORTCUT.md` Changes

1. Update `docs/SHORTCUT.md` first.
2. Run `python3 docs/diagrams/generate_shortcut_quickstart.py`.
3. If the English image looks correct but the Chinese image falls back to English text, the `ZH_TEXT` mapping in the script is incomplete.
4. Update `SECTION_TITLE_ZH` and `ZH_TEXT` in `docs/diagrams/generate_shortcut_quickstart.py`, then run the script again.
5. Confirm that both images have been updated.

## Design Prompt

If you want to keep the information structure but continue refining the visual style, use the following design prompt:

```text
Create a vertical shortcut quickstart image for zmux that helps beginners get started quickly. Follow the visual direction in docs/diagrams/shortcut-quickstart-philosophy.md: dark terminal styling, calm and restrained tone, structure-first presentation, a two-column console-style card layout, and low-noise high-readability design. Highlight Prefix = Ctrl+a at the top, then organize the rest into Pane / Window / Session / Command Line / Copy Mode / Shell Pass-through / Notes sections. The image should not feel like a poster. It should feel like a practical quick-reference map that a beginner can scan in one glance. Do not turn the entire `docs/SHORTCUT.md` file into a long, flat, line-by-line infographic. Emphasize the most frequently used shortcuts, keep the grouping clear, preserve strong visual hierarchy, and make the output maintainable over time.
```

## Maintenance Notes

- If only wording changes, update `docs/SHORTCUT.md` first and then rerun the script.
- If a completely new category is added, update the section order and Chinese section titles in the script before rerunning.
- If English text appears in the Chinese image, `ZH_TEXT` is missing translations and must be completed before regenerating.
- Do not delete the generation script, or the output will no longer be reproducible in a stable way.
